# Portfólio — Matheus Teixeira Francelino (Next.js + React)

Inspirado no estilo/estrutura de bartbeyond.art (layout brutal, tipografia mono, cards e CTA com personalidade).

## Rodar local
```bash
npm install
npm run dev
```

## Onde editar
- Conteúdo (projetos/skills): `lib/data.ts`
- Links de contato / e-mail / GitHub / LinkedIn: `components/contact.tsx`
- Páginas:
  - Home: `app/page.tsx`
  - Builds: `app/builds/page.tsx`
  - About: `app/about/page.tsx`
